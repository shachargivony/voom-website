# VOOM Website

אתר VOOM - שיווק דיגיטלי וקידום עסקים

## התקנה

```bash
npm install
```

## הרצת שרת פיתוח

```bash
npm run dev
```

האתר יפתח ב- `http://localhost:5173`

## בניית גרסת ייצור

```bash
npm run build
```

## טכנולוגיות

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- Radix UI

## מבנה הפרויקט

```
src/
├── components/     # רכיבי React
├── pages/          # דפי האתר
├── api/            # API clients
├── lib/            # פונקציות עזר
└── components/ui/  # רכיבי UI בסיסיים
```
